Para compilar el codigo  Primero nos colocaremos en la ubicación de archivo en la consola,  se usará el comando javac Control.java
Una vez compilado para ejecutar el codigo se usará el comando java Control

El programa consiste en la resolución de problemas usando metodos estáticos donde emplearemos variables, arreglos y en uno usaremos listas 
como medio para su respectiva respuesta
El la prueba del problema 6 agregue una impresion de pantalla para mostrar la lista generada por el metodo, la cual puede varias cambiando 
el numero n en el metododo problema6(n).
